## Publishing to Packagist

This package is published to [Packagist](https://packagist.org/packages/laravel-mailjet/laravel-mailjet) automatically via a [GitHub webhook](https://packagist.org/about#how-to-update-packages). When a new release is tagged on GitHub, Packagist is notified and updates the package index automatically.
